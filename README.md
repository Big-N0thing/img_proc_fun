# IMGPROG

## Procesamiento de imágenes
### Features propuestas:
- Rotar imagenes
- Pasar a escala de grises
- Pasar a blanco y negro
- Pasar a ASCII
- Filtrado de imagen
- Estadística de imagen (p.ej.: histogramas)
